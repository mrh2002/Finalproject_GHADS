*** Gaza Humanitarian Aid Distribution System (GHADS) ***

*** Project Overview ..

GHADS (Gaza Humanitarian Aid Distribution System) is a desktop application developed using Java to help humanitarian organizations coordinate aid distribution for displaced families in Gaza.

The system provides a centralized database that allows multiple organizations to work together efficiently and transparently while preventing duplicate aid distributions.

--------------------------------------------------------------------------------

*** Problem Statement ..

When humanitarian organizations work independently, the same family may receive aid multiple times while other families receive no assistance at all.

GHADS solves this problem by maintaining a shared database that tracks all families and aid distributions, ensuring fair and transparent aid delivery.

--------------------------------------------------------------------------------

*** Project Goals ..

- Register displaced families in one centralized database.
- Record all aid distributions.
- Prevent duplicate aid distributions.
- Identify families that have not received aid.
- Support role-based access control.
- Provide dashboards and statistical reports.
--------------------------------------------------------------------------------

- Java
- JavaFX

--------------------------------------------------------------------------------
*** System Architecture ..

The system follows:

** MVC Pattern:

- Model
- View
- Controller

** DAO Pattern:
Used for all database operations.

** Singleton Pattern:
Used for database connection management.
